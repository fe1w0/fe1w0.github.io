SQL_16 [二次注入]

注册页面存在三个参数:

username password age

其中age只能是Number,UserName为4-16 bit a-zA-Z0-9_- !

但age会支持0x进制来表示,同时在succ.php中可以解码

如age=0x276f7220313d3123

所以打算从这点开始入手,pass和user部分测了半天..啥的没有

同时发现,age 有个再次与数据库中的数据进行查询比较的过程,即同岁功能。

注册时,设置age为1 and 0 union select 1,2,3,4,5#,16进制为0x3120616e64203020756e696f6e2073656c6563742020312c322c332c342c3523
